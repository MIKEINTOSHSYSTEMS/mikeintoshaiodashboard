About this Guide

This User Guide outlines the process for creating and managing HCT quotes for Direct and Affiliate Sales using Microsoft CRM and SAP. The quotes-related features of CRM are currently only available within Halliburton Completion Tools (HCT).

Prerequisites

The following training courses MUST be completed before requesting access to GQS. These courses are available on Halliburton University.

  • Export Compliance
  • Code of Business Conduct
  • Business Acquisition Process
  • Internal Controls

This guide assumes that you already have access and are familiar with how to use Microsoft CRM to view/manage opportunities. If you are not familiar with Microsoft CRM, please access and review the CRM Training for BD course on Halliburton University.

Getting Started

Log in to GQS

To log in to GQS, open your browser, type CRM and press enter. Initial log on to CRM displays the Workplace view. To navigate to Quotes:

  1. Click Sales.
  2. Click Quotes.

Set default view

  1. Click File > Options. The Set Personal Options screen opens.
  2. Click the General tab.
  3. In the Default Pane dropdown menu, select Sales.
  4. In the Default Tab dropdown menu, select Quotes.
